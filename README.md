# Essential Process Maps for Sample Registration Systems (SRS)

An interactive toolkit for exploring essential **Sample Registration System (SRS)** workflows through BPMN-based process maps.

**Live toolkit:** <https://swisstph.github.io/srs-process-navigator/>  
**Date:** June 2026  
**Version:** 1.0
---

## Overview

This toolkit provides generic process maps for key SRS workflows. It supports planning, implementation, review, and improvement of SRS processes.

The maps are intended as a **reference model** and should be adapted to the local context, including institutional roles, resources, geography, and system maturity.

---

## Features

- Interactive SRS process navigator
- BPMN-based process maps
- Process descriptions and operational guidance
- Export options for maps and documentation

---

## How to use

Open the live toolkit and select **Open navigator**. Choose a subprocess to view its process map and related guidance.

---

## Deployment

This is a static HTML toolkit published with GitHub Pages.
